# DynBERG

This repository contains the implementation of the DynBERG model, designed for financial fraud detection in dynamically evolving financial transaction graphs. The project is built using PyTorch, ensuring modularity, scalability, and ease of integration into existing workflows

## Requirements
Requires Python 3.10 or later with PyTorch and related libraries. Please refer to requirements.txt for details of python packages required.
